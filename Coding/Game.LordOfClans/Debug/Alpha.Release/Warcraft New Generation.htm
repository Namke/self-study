Current usage

 

    Cấu hình bằng parameter:

        War.Exe M* F*

        M* cấu hình độ phân giải màn hình: M1024 cho độ phân giải 1024x768, M800 cho độ phân giải 800x600, chuẩn là M1024

        F* cấu hình chế độ chạy: F+ cho chạy toàn màn hình, F- cho chạy chế độ cửa sổ, chuẩn là F+

    Kiểm tra thông tin debug, cheat (Nhập ở console khi chơi game):

        ShowClansInfo : Hiển thị thông tin về các clan.

        ShowMapInfo : Hiển thị thông tin về bản đồ hiện tại.

        ShowGameInfo : Hiển thị thông tin chung về game.

        ShowGlobalInfo : Hiển thị thông tin chung về game.

        ShowUnitInfo : Hiển thị thông tin về đơn vị đang chọn.

        ShowScriptDebug : Hiển thị thông tin debug về kịch bản đang chạy.

        NoFog : Tắt sương mù.

        OnScreen : Hiện toàn bản đồ.

        PointBreak : Không giới hạn đơn vị.

        WhoNeedMoney? : Thêm tiền.

 

    Phím tắt:

        Thoát khỏi game : Alt+X.

        Điều chỉnh tốc độ game: Nút + và - ở numpad.

        Chọn nhóm quân bằng kéo chuột, chọn thêm một đơn vị hoặc loại một đơn vị đã chọn khỏi nhóm bằng Control + kích chuột trái vào đơn vị đó.

        Huỷ lệnh bằng phím phải chuột hoặc nút Cancel (Nếu có).

        Ghi nhớ nhóm quân Control + phím số # (0-9), chọn nhóm quân đã nhớ bằng phím số.

        Phím tắt để gọi nút có thể xem ở bảng tooltip.

Current usage

 

    Configuration by parameter:

        War.Exe M* F*

        M* config video resolution: M1024 about res 1024x768, M800 about res 800x600, default is M1024

        F* config run mode: F+ to full screen mode, F- for windowed mode, default is F+

    Debug checking, cheat code (Enter in console when play game):

        ShowClansInfo : Show information about clans.

        ShowMapInfo : Show information about current map.

        ShowGameInfo : Show global information about game.

        ShowGlobalInfo : Show global information about game.

        ShowUnitInfo : Show information about current unit selected.

        ShowScriptDebug : Show script debug info.

        NoFog : Disable/enable fog.

        OnScreen : Show all map.

        PointBreak : Unlimit unit.

        WhoNeedMoney? : Add more money.
